# Candy-Distribution
US Candy Distribution (PBI)
# Why?

This Power BI Report builds looks at a combination of 5 csv files tied to Candy Distribution in the US and Canada from 2021 to 2022.

# The Data

* A zip file containing 5 CSV files cover the data for this report.
   - A factories file that looks at the locations of where the candy products are manufactured
   - A products file which details the cost of each product, their name and other relevant data
   - A sales file which is the main fact table with all relevant customer ID's, metrics for sales, regions and so on
   - A targets file that looks at the different candy categories (3) and the sales expectations
   - A us zips file for all the cities and their relevation locations via longitude and latitude
